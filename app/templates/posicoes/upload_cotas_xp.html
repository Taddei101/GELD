Upload de Posições XP

Selecione o arquivo Excel da XP para importar suas posições em fundos de investimento.

O sistema irá:

  • Processar a aba "Sua carteira" do arquivo XP
  • Identificar automaticamente os fundos de investimento
  • Verificar se os fundos já existem no sistema
  • Cadastrar novos fundos conforme necessário
  • Registrar as posições para o cliente

Nota: Como o arquivo da XP não contém CNPJs, o sistema tentará fazer correspondência com fundos existentes pelo nome.
